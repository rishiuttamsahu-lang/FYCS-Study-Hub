========================================================================
                      FYCS STUDY HUB - BUG REPORT
========================================================================

Here is the documentation of the major issues encountered in the FYCS Study Hub project (including both web and Android app wrappers).

------------------------------------------------------------------------
Issue 1: Mobile Share-to-App Redirection (Cold Start vs. Hot Start)
------------------------------------------------------------------------
- Problem Definition: 
  * Hot Start (App already running in background): Sharing a document from mobile file explorer to the app works fine, transferring the user to the Upload page.
  * Cold Start (App completely closed/killed): Sharing a document from mobile file explorer launches the app but leaves it on the main Home page instead of navigating to the Upload page.

------------------------------------------------------------------------
Issue 2: Google Drive Picker Crash inside App (OAuth2 Undefined)
------------------------------------------------------------------------
- Problem Definition:
  Clicking the Google Drive button inside the mobile app throws a TypeError crash: "Cannot read properties of undefined (reading 'oauth2')".

------------------------------------------------------------------------
Issue 3: Flash of Login Page on Slow Networks (Login Session Delay)
------------------------------------------------------------------------
- Problem Definition:
  On slow networks, users who are already logged in temporarily see the Login page for 1-2 seconds, which then disappears and redirects them back to the Home page.

========================================================================
